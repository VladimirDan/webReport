Завдання 1

URL Scheme в JavaScript

В JavaScript схема URL є вбудованим об'єктом, який дозволяє розбити URL-адресу на її складові частини. Давайте розглянемо простий приклад використання цієї схеми:

const urlString = "https://www.facebook.com";

Створення об'єкту URL за допомогою URL-адреси

const url = new URL(urlString);

Отримання різних компонентів URL

const protocol = url.protocol; const hostname = url.hostname; const pathname = url.pathname; const searchParams = url.searchParams; const hash = url.hash;

Виведення компонентів URL у консоль

console.log("Протокол:", protocol); console.log("Доменне ім'я:", hostname); console.log("Шлях:", pathname);

У цьому прикладі ми створюємо об'єкт URL з рядка urlString а потім отримуємо доступ до різних компонентів URL, таких як протокол, доменне ім'я, шлях. Усі ці компоненти можна окремо використовувати для подальшої обробки в JavaScript-коді.
