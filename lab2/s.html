селектори

  1. Тегові селектори: вказують на HTML-елементи за назвою тегу.
    p { color: blue; }
  2. Класові селектори: вказують на елементи за назвою класу.
    .text-highlight { background-color: yellow; }
  3. Ідентифікаторні селектори: вказують на унікальний елемент за його ідентифікатором.
    #unique-text { font-weight: bold; }
  4. Атрибутні селектори: вибирають елементи за наявністю або значенням їх атрибутів.
    input[type="text"] { background-color: #f0f0f0; }
  5. Псевдокласові селектори: вибирають елементи в залежності від їх стану.
    a:hover { color: red; }
  6. Псевдоелементні селектори: використовуються для стилізації частин елементів.
    p::first-letter { font-size: 200%; }
