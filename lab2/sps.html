Способи підключення стилів CSS

  1. Внутрішній CSS (Inline CSS) - стилі вписуються прямо в атрибут style HTML-елемента.
    <p style="color: red;">Текст</p>
  2. Вбудовані стилі (Internal CSS) - стилі розміщуються у тегу <style> в <head> секції документа. 
    <style>
    p { color: red; }
</style>
  3. Зовнішні стилі (External CSS) - стилі зберігаються в окремому файлі .css, який підключається за допомогою <link>.
    <link rel="stylesheet" href="style.css">
  4. Імпорт CSS - використання @import у CSS-файлі для підключення інших CSS-файлів.
    @import url("style.css");
